# control de acceso con RFID
### Este es un proyecto de un control de acceso con un lector RFID, utilizando un ESP32 como microcontrolador.
### Los componentes utilizados son:
1. RFID RC522
2. ESP32 dev kit v1
3. lcd display 16x5
4. Placa 10x7cm
5. aa
